#  Local LLM App with Ollama + Gemma3:270M

Ứng dụng này cho phép bạn **tương tác với mô hình ngôn ngữ lớn (LLM)** ngay trên máy tính cá nhân **không cần kết nối internet**, sử dụng **[Ollama](https://ollama.ai/)** và **mô hình Gemma3:270m** – phiên bản nhẹ, chạy tốt trên máy có RAM từ 8GB.

---

##  Tính năng

Chạy mô hình LLM cục bộ (không cần OpenAI API)  
Hỗ trợ giao diện web bằng **Streamlit**  
Cho phép nhập prompt, chọn mô hình, và hiển thị kết quả ngay  
Hỗ trợ các tác vụ:
- Tóm tắt văn bản  
- Dịch câu  
- Sinh nội dung tự động  
- Giải thích, phân tích, trích xuất thông tin
